# Hunter

[Hunter](https://github.com/ikuer/Hunter.git)：关注每日焦点资讯，主要涵盖时政、经济、科技、技术信息、数据和城市发展规划等方面。

## 总则

1. 更快 ・ 更简单。
2. 保持专注。
3. 有价值资讯和重要事件。

## 命名规则

- 文件夹：年份（2016）管理月份（05）
- 每月资讯归档为一个文件（当前月份）

```
Hunter/
  +- 2016/
    +- 05.md
  +- 2017/
    +- 01.md
    +- 02.md
  +- README.md
```

## 内容规范

1. 资讯标题
2. 链接
3. 内容摘要
